## Key learnings (this build)
- **NO autoplay / no compute bomb** (Felipe, grave): animations default paused, run once, stop on hidden tab. `useAnimator`.
- **Animation only where motion teaches** (owner's call): keep for fronts/waves/plumes; static heatmap elsewhere.
- **ODE PINN needs SOFT IC**: a `t¬≤` hard-IC ansatz kills `‚àÇŒ∏ÃÇ/‚àÇparams` near t=0 ‚Üí the net never learns the initial
  acceleration (moved Œ∏‚ÇÅ the wrong way). Soft IC (IC + OperatorBC, weighted 100√ó) ‚Üí tracks the chaotic ODE (0.09%
  over 1.5s; leave-time 1.99s over 3s). tanh > SIREN for the soft-IC IVP.
- A **trajectory is a 1-D-in-t multi-output trace** ‚Äî no new schema; the 1-D trace keeps up to 601 samples;
  `extra_metrics(sf)` injects the RK45 anchor + twin-IC as extra baked fields; `web_drivable=False` ‚Üí precompute.
